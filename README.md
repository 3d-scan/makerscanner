makerscanner
============

Software for the makerscanner.  See http://www.makerscanner.com/ for details and  instructions.


To build from source on Linux:

    sudo apt-get install libwxgtk2.8-dev
    (or equivalent for your system)

    autoscan
    autoheader
    aclocal
    autoconf
    automake -a
    ./configure
    make
    sudo make install
